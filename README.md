# 📱 スマホ操作れんしゅう教材

高齢者や初心者向けに、スマホ操作を楽しく練習できる教材です。  
タップ・ダブルタップ・長押し・スワイプ・ピンチイン/アウトをゲーム感覚で習得できます。
